The Portfolio Mapping Tool (PMT) Version History

The Portfolio Mapping Tool 3.0 (PMT 3.0) is a web-based mapping application intended to visualize development work in a variety of ways. The features of this web application allow users to browse, map, add, edit, and share projects without the need for geographic information systems expertise. This Version History provides the lastest information on updates, features changes, bug fixes and release dates for the PMT 3.0.

3.10.4

2/24/17

Updates

  • Added geographic filter to the Map page. The geographic filter provides a filter menu of one or more boundary levels. Selecting a boundary feature will limit the data on the map to only those activities which have locations within the selected boundary features.
  • Added geographic filter to the Activity page. The geographic filter provides a filter menu of one or more boundary levels. Selecting a boundary feature will limit the activities listed to only those activities which have locations within the selected boundary features.
  • Expanded the data model so that locations are mapping level aware. This means that any given location in the database understands what administrative level it is map to and how it is related to any boundary loaded with in the system. Locaiton mapped at the country level will not be shown or aggregated below the admin level zero of any boundary.
  • Added configurable "in-use" option to all taxonomy filters (Summmary, Map & Activity pages). When enabled, only classifications in-use by the selected data groups will be shown in the filter options. The filter options are updated when the data group selections are updated.
  • Removed un-used elements from the API end-point serving the Activitiy page listing to improve performance.
  • Added Institutional contextual layers to the Map Page (EthAIM Only).
  • Added Livestock summary charts to National and Regional areas of the Summary Page (EthAIM Only).
  • Update the point symbology configuration and rendering for contextual layers.
  • Update vertical chart directive to automatically color "Other" summary counts grey and remove the "Other" from legend when "Other" counts were not present.
